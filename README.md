# Nitesh Raj — Chinese Translator & Interpreter Website

A professional, SEO-optimized single-page website built to generate direct leads for a Chinese (Mandarin) Translator & Interpreter. The site is designed for clarity, trust, and conversion, targeting individuals, companies, and agencies looking for reliable language services.

---

## Purpose

This website serves as the official digital presence of **Nitesh Raj**, enabling potential clients to:

- Understand offered translation and interpretation services
- Verify credibility through past work and testimonials
- Quickly initiate contact via form, WhatsApp, or email

The primary goal is **lead generation**, not portfolio display.

---

## Tech Stack

- **Frontend**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **SEO**: React Helmet
- **Animations**: Lightweight CSS and JS-based transitions
- **Deployment**: Static hosting compatible (Vercel, Netlify, Cloudflare Pages)

---

## Design Philosophy

- **Color palette**: Teal / green (primary), white (backgrounds), grey (text & accents)
- **Style**: Clean, corporate, trustworthy
- **UX focus**: Clear messaging, minimal friction, fast load times
- **Responsive**: Optimized for desktop, tablet, and mobile devices

## SEO & Accessibility

- Semantic HTML structure (`header`, `main`, `section`, `footer`)
- Proper heading hierarchy (H1–H3)
- Keyword-focused content targeting:
  - Chinese Translator
  - Chinese Interpreter in India
  - Mandarin Translator for Business
- Optimized meta title and description
- Accessible form labels and contrast-safe colors

## Getting Started

### Prerequisites
- Node.js (v18 or later recommended)
- npm or yarn

### Install Dependencies
```bash
npm install
```
## Run Development Server
```bash
npm run dev
```

## Build for Production
```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```
